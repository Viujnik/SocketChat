# SocketChat
Это небольшой проект преднозначенный для создания чего-то в духе консольного чата. Но написан проект с помощью работы с сокетами, был выбран более тяжелый путь(хотя, учитывая мииниатюрность проекта, слово 'тяжелый' подходит с трудом).

## Что за чат
Нет смысла вдаваться в техническое объяснения, что такое сокеты и т.п. Но, если вкратце - есть сокет сервера, который принимает сокеты клиентов. Всё это разбавленно щепоткой многопоточности. Все клиенты храняться в словаре из std::map, определены над ним операции add/update/remove client. Можете почитать для больших подробностей, но спешу огорчить - ничего нового вы тут не найдете.)
